Zooko and Seth in RSS

Since I’m spending more and more time in NetNewsWire, I decided to RSSify the few remaining sites I read. First off I did Zooko’s website: RSS feed. If you don’t read Zooko’s blog, the source code, while very messy, might still be useful for scraping a site you like.

I modified it to create another scraper (source) for Vitanuova, which has become my absolute favorite living web site. It doesn’t do much, but it does let my aggregator notify me when there’s an update, which is nice.
