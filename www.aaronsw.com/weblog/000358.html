cloudmark

The folks behind the free-software razor have teamed up started a new company: cloudmark. They’ve build Cloudmark SpamNet, a plugin for Outlook that lets Windows users get in on the spam-filtering fun. Now all they need to do is build in SpamAssassin…

Wait, Deersoft Inc. is developing SpamAssassin Pro, an Outlook plugin that uses SpamAssassin.
