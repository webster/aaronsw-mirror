How to Do Stuff

I’m proud to introduce my new series: How to Do Stuff. It currently features two articles:

  • How to Do Version Control with CVS
  • How to Do Live Streaming

I’m especially proud of How to Do Live Streaming. It covers how to do broadcasting, reflection, encoding, recording, etc., all using pipes of small Unix tools.

The whole section is written using my humane structured text format, atx. You can see the source code of a page by adding .atx to the end.

(These Movable Type entries are also written in atx. I wrote a small Movable Type plugin to pass the entries through my atx converter before publishing.)
