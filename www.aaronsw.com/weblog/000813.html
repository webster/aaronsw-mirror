Fascinating User Interface Research

Ka-Ping Yee: Peephole Displays. This most innovative UI ideas per minute I’ve ever seen: Lists you can select from using spacial memory. Maps you can zoom by moving up and down. And coolest of all, a 2-dimensional clipboard that you copy into by pulling something up and paste by punching it down. Awesome, awesome stuff. Watch the first video and the end of the second to get a quick idea.

The paper includes a list of unimplemented ideas: 3D folder navigation (with file-moving using 3D drag and drop), using different lanes for comparing versions of a changing document, and a Web browser. Drool.

Only problem: lots of papers but no code!
